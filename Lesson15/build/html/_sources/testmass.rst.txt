Модуль TestMassConnection
=================================================

Модуль автоматизированного запуска сервера и нескольких клиентов

Схема запуска:

После запуска выводится описание доступных команд.
Допустимые команды:

1. s - Запустить сервер
 
* Запускает сервер с ip и портом по умолчанию.

2. x - Остановить сервер

* Завершает работу сервера.
 
3. [X] - Запустить X клиентов
 
* Будет запущено Х клиентов (где Х - целое число больше 0)).
 
4. y - Остановить всех клиентов
 
* Завершает работу всех запущенных клиентов.
 
4. q - Остановить сервер, клиентов и выйти
 
* Завершает работу всех клиентов, сервера и закрывает модуль.
