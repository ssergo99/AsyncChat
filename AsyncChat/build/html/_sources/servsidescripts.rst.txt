Набор вспомогательных скриптов сервера
======================================

Набор скриптов, использующихся для работы серверной части проекта.

server_side.py
--------------

Запускаемый модуль,содержит функционал инициализации и работы серверной части.

server_side. **parsing_arguments** ()
    Парсер аргументов командной строки, возвращает кортеж из 2 элементов:

    * адрес на котором запустить сервер
    * порт на котором запустить сервер

server_side. **start_serv** ()
    Основная функция, запускающая сервер,
    содержит подфункции:
        создания/загрузки параметров конфигурации из ini файла
        запуска графической оболочки администратора сервера
        создания объекта класса Server
